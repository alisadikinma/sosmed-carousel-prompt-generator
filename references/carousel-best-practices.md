## 2. Universal Carousel Structure (All Platforms)

### The HOOK â†’ VALUE â†’ CTA Framework

```
Slide 1: HOOK â€” Stop the scroll (curiosity gap, bold claim, shocking stat)
Slide 2: CONTEXT â€” Set up the problem or promise
Slide 3-N: VALUE â€” Deliver the content (tips, steps, data, insights)

```

### Slide Sequence Psychology

| Slide | Purpose | Engagement Trigger |
|-------|---------|-------------------|
| 1 (Hook) | Stop scrolling | Curiosity gap, pattern interrupt |
| 2 (Setup) | Create investment | Problem identification, "this is for you" |
| 3-7 (Value) | Deliver dopamine hits | One idea per slide, progressive reveal |
| 8-9 (Peak) | Strongest content | Most shareable insight, "save this" moment |
| Last (CTA) | Convert attention | Follow, save, share, comment, link |

### The "One Idea Per Slide" Rule
- Each slide = ONE concept, ONE visual, ONE takeaway
- If you need a paragraph, it's too much â€” split into 2 slides
- White space is your friend â€” don't crowd the frame
- Bold the KEY WORD in each slide (accent color highlight)

## 3. Instagram Carousel Best Practices

### Algorithm Signals
- **Saves** are the #1 ranking signal for carousels (more than likes)
- **Shares** are #2 â€” design for "tag a friend who needs this"
- **Dwell time** matters â€” more slides = more time on post = better reach
- **Completion rate** â€” Instagram tracks how many slides people swipe through
- **Re-visits** â€” carousels that people come back to get boosted

### Design Rules
| Rule | Why |
|------|-----|
| First slide = no text-heavy design | Feed preview must be visually striking |
| Use 4:5 aspect ratio for feed carousels | Takes up more screen real estate than 1:1 |
| Consistent color palette across all slides | Brand recognition + professional look |
| "SWIPE â†’" indicator on all slides except CTA | Increases swipe-through rate by 20-30% |
| Number your slides ("3/8") | Creates completion drive |

### Content Types That Perform Best

4. **Myth vs Reality** â€” Contrarian takes
5. **Data storytelling** â€” Stats with visual context
6. **Mini case studies** â€” Real results, real numbers

### Caption Strategy
- First line = hook (matches slide 1 hook)
- Body = expand on carousel content (not repeat it)
- End with question or micro-CTA
- 5 hashtags max (in caption, not comments) â€” Instagram algorithm penalizes >5 since Dec 2025

### Posting Cadence

- Best times: 7-9 AM, 12-1 PM, 7-9 PM (local timezone)
- Tuesday-Thursday highest engagement
- Avoid posting within 2 hours of a previous post

## 8. Visual Design Best Practices

### Typography Hierarchy
```
HEADLINE: 28-36pt, Bold, ALL CAPS (accent color for power word)
Subhead:  18-22pt, Semibold, Mixed case
Body:     14-16pt, Regular, Mixed case
Caption:  12-14pt, Light, Mixed case
```

### Color Usage
- **Background**: Dark (better readability, premium feel) or clean white
- **Primary text**: White on dark bg, dark gray on light bg
- **Accent**: ONE accent color for keywords/highlights (brand color)
- **Avoid**: More than 3 colors per slide, neon on white, low contrast

### Layout Patterns
| Pattern | Best For | Description |
|---------|----------|-------------|
| **Centered stack** | Tips, quotes | Text stacked vertically, centered |
| **Split panel** | Before/after, compare | Left-right or top-bottom division |
| **Full bleed image** | Hook, thumbnail | Image fills frame, text overlay in gradient zone |
| **Icon + text** | Listicles, steps | Icon left, text right (or icon top, text bottom) |
| **Data card** | Stats, numbers | Large number + context label |

### Image Integration
- Creator face: 40-60% of frame on Hook/CTA slides
- B-roll: full frame, no creator face
- Gradient overlay: bottom 30-40% for text readability
- Consistent image style across all slides (same filter, same lighting)
